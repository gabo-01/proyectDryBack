Proyecto de Machine Learning para el Cálculo y Clasificación de Humedad en Procesos de Secado Metalúrgico

Este proyecto constituye la base de un sistema inteligente orientado al análisis y optimización del proceso de secado en la industria metalúrgica. Abarca de forma integral todas las etapas necesarias para el desarrollo de modelos de Machine Learning aplicados a la predicción y clasificación de niveles de humedad en muestras procesadas.

Desde la ingestión de datos a partir de archivos CSV y su posterior almacenamiento estructurado en bases de datos SQL, hasta un análisis exploratorio detallado que permitió una limpieza profunda y validación de la información, se garantizó la calidad de los datos empleados en el entrenamiento.

Se implementaron dos modelos clave:
	•	Modelo de regresión, para predecir con alta precisión el nivel exacto de humedad.
	•	Modelo de clasificación, para identificar si una muestra requiere un nuevo ciclo de secado.

Finalmente, se desarrollaron endpoints REST que permiten el consumo de estos modelos desde una aplicación móvil, facilitando la toma de decisiones en tiempo real en ambientes industriales.

Este sistema representa un paso significativo hacia la digitalización inteligente de procesos metalúrgicos, ofreciendo eficiencia, precisión y escalabilidad.
